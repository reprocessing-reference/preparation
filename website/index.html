The processing of Sentinel 1,2,3 Level-1/2 products uses a set of auxiliary files to ensure that the generated products meet the quality and calibration/validation specifications.

The Reprocessing Auxiliary Service is comprising of:

  • Reprocessing Configuration Baseline providing for a period of time the list of auxiliary files applicable for the generation of Level-1/2 products
  • Reprocessing Data Baseline providing the list of applicable auxiliary files associated with each L0 product (needed for the generation of each Level-1/2 products.
  • AUXIP: the interface point for downloading the auxiliary data files.
