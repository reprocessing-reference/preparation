News    :    March 2022    :    The processing baseline 03.40 for Sentinel-1 and the processing baseline 04.00 for Sentinel-2 are ready.

 

The processing of Sentinel 1,2,3 Level-1/2 products uses a set of auxiliary files to ensure that the generated products meet the quality and calibration/validation specifications.

The Reprocessing Auxiliary Service is comprising of:

  • Reprocessing Configuration Baseline implements an API allowing to query and retrieve the up-to-date list of applicable auxiliary files (excluding dynamic auxiliary files) given a mission, satellite unit, sensing time period and product type to be generated, without any application of selection rules.
  • Reprocessing Data Baseline, given a L0 product and a product type to be generated, identifies the applicable set of auxiliary files (static and dynamic) after application of the selection rules identified for each auxiliary type.
  • AUXIP: the interface point for downloading the auxiliary data files.
